# Linux File Permissions Project

This project demonstrates how to analyze and modify file and directory permissions in a Linux environment to improve system security.

## Objective

To ensure that file permissions align with organizational security policies by restricting unauthorized access.

## Scenario

A research team needed to update file and directory permissions within a project directory. Some files had overly permissive access, which posed a security risk.
